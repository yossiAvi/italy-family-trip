# שיתוף מיקום משפחתי – גרסה 2.2.0

נוסף אזור "איפה כולם?" עם שיתוף מיקום מרצון בזמן שהאתר פעיל.

## הפעלה
1. העלו את קוד המקור ל-GitHub/Netlify.
2. פתחו Supabase > SQL Editor.
3. הריצו שוב את `supabase/storyline.sql` המלא. הקובץ משתמש ב-`if not exists`, ולכן בטוח להריץ אותו גם על פרויקט קיים.
4. ודאו ש-Anonymous Sign-ins פעיל.

## פרטיות ומגבלות
- נשמר רק המיקום האחרון של כל בן משפחה, ללא היסטוריית מסלול.
- השיתוף מתחיל רק לאחר לחיצה ואישור הרשאת המיקום בדפדפן.
- ניתן לעצור או למחוק את המיקום בכל עת.
- בדפדפנים ניידים העדכון עלול להיעצר כאשר המסך כבוי או כשהאתר נמצא ברקע.
