# הפעלת Story Line משפחתי בענן

האפליקציה עובדת מיד במצב מקומי. במצב הזה התמונות והסיפורים נשמרים רק בדפדפן שבו הוזנו.
כדי שכל המשפחה תראה ותוסיף לאותו יומן מכל טלפון, מחברים פרויקט Supabase חינמי.

## 1. יצירת פרויקט Supabase

1. היכנסו ל־Supabase וצרו פרויקט חדש.
2. ב־Authentication → Providers → Anonymous Sign-Ins הפעילו כניסה אנונימית.
3. פתחו SQL Editor.
4. העתיקו והריצו את הקובץ `supabase/storyline.sql`.

## 2. קבלת מפתחות

ב־Project Settings → API העתיקו:

- Project URL
- Publishable/anon key

המפתח שמוזן באתר הוא מפתח ציבורי. אין להשתמש ב־service_role key בדפדפן.

## 3. הגדרה ב־Netlify

באתר שלכם ב־Netlify:

1. Site configuration → Environment variables.
2. הוסיפו:

```text
VITE_SUPABASE_URL=https://YOUR-PROJECT.supabase.co
VITE_SUPABASE_ANON_KEY=YOUR-PUBLISHABLE-KEY
VITE_TRIP_ID=italy-family-trip-2026
VITE_STORY_BUCKET=trip-story-images
```

3. ודאו שהמשתנים זמינים ל־Builds.
4. הפעילו Deploy חדש באמצעות Clear cache and deploy site.

## 4. בדיקה

1. פתחו את האתר משני טלפונים שונים.
2. הוסיפו סיפור מהטלפון הראשון.
3. הסיפור אמור להופיע בטלפון השני בתוך שניות.

## אבטחה והתנהגות

- כל מכשיר נכנס כמשתמש אנונימי ייחודי.
- כל בני המשפחה יכולים לצפות בכל הרשומות.
- כל משתמש יכול לערוך או למחוק רק רשומות שהוסיף מהמכשיר שלו.
- מי שמקבל את כתובת האתר יכול לצפות ולהוסיף. מומלץ לשמור את כתובת האתר בתוך המשפחה בלבד.
- ניתן בהמשך לשדרג לכניסה בקוד חד־פעמי במייל או לרשימת מוזמנים סגורה.

## הרשאות בטלפון

בעת השימוש הראשון, הדפדפן יבקש הרשאה למיקום, מצלמה ומיקרופון. יש לבחור “אפשר” כדי להשתמש בזיהוי מיקום, צילום והכתבה קולית.
