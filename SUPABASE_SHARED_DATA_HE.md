# סנכרון כל הנתונים המשפחתיים ב-Supabase

גרסה 2.0 מוסיפה סנכרון בזמן אמת של:

- תקציב מתוכנן ויומן הוצאות
- פתקים ופרטי קשר
- צ'ק-ליסט ופריטים אישיים
- כרטיס חניה ותמונת החניה
- דירוגים משפחתיים
- משימות הילדים
- משפטים שמורים במתרגם
- Story Line ותמונות (כפי שכבר פעל קודם)

## הפעלה חד-פעמית

1. פתחו Supabase Dashboard.
2. עברו אל SQL Editor.
3. הריצו מחדש את הקובץ `supabase/storyline.sql` המלא.
4. ודאו שב-Authentication > Providers האפשרות Anonymous Sign-ins פעילה.
5. בצעו Deploy חדש ב-Netlify.

הסקריפט בטוח להרצה חוזרת ומשתמש ב-`if not exists` וב-`drop policy if exists`.

## מעבר ממידע מקומי

בכניסה הראשונה, אם עדיין אין ערך בענן, האתר מעביר אוטומטית את המידע המקומי הקיים אל Supabase. לאחר מכן כל שינוי מסתנכרן לכל המכשירים בזמן אמת.
