# גרסה 2.1.0 — שיפור קריאות וממשק

- סינוני אוכל וקניות הוחלפו מתצוגת טאבים ארוכה לתפריטים נפתחים.
- קטגוריות הצ׳ק־ליסט הועברו לבורר קומפקטי.
- כלי הטיול מוצגים כגריד כרטיסים נוח בנייד.
- כלי החוויה המשפחתית מוצגים בגריד ללא גלילה אופקית.
- קטגוריות המתרגם נשברות לגריד מסודר.
- תוקנו צבעי טקסט וניגודיות בכרטיסים, שדות וסכומים.
- נוספו מצבי focus ברורים לנגישות מקלדת.
- בנייד הוגדלו שדות הבחירה והכפתורים.
